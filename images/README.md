# Images Folder

## Headshot
Place your professional headshot here as:
- **Filename:** `husni-rifqi.jpg`
- **Recommended size:** 800×900px or similar portrait ratio (4:4.5)
- **Format:** JPEG (JPG), optimized for web (~100–200KB)

The site will display a placeholder if the image is not found.

## How to add your photo
1. Save your headshot as `husni-rifqi.jpg` in this folder
2. Commit and push the change
3. The photo will automatically appear in the Hero section

## Open Graph Image
For best social media sharing previews, ensure `husni-rifqi.jpg` is present.
The `og:image` tag in index.html references this file.
