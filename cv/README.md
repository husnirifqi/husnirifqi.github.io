# CV Folder

Place your CV PDF here as:
- **Filename:** `CV_Husni_Rifqi.pdf`

The "View CV" and "Download CV" buttons in index.html reference this file.

## Updating your CV
1. Export your latest CV as a PDF
2. Name it `CV_Husni_Rifqi.pdf`
3. Replace the old file in this folder
4. Commit and push — the download button will automatically serve the new version
